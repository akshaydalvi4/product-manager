=== Plugin Name ===

Dropudown Frontpage Plugins 

== Description ==

Plugin to load different types of menus with pictures this Dropdowns are displayed on the frontend of your site or redirection.


== Features ==

Create unlimited dropdowns
Display on page using the included shortcode eg. [Akshay-dropdown]
Redirection method also available

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
